## Description

**vscode-feeds** is a Visual Studio Code extension that brings social media feeds (such as Twitter) directly into your editor. Stay updated with the latest posts, trends, and discussions without leaving your coding environment.

## Usage

- Open the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
- Search for commands related to `vscode-feeds`
- Follow the prompts to connect and view your feeds
